## 🎯 Quick Overview

This Power BI dashboard explores Starbucks beverage data and helps answer questions like:

☕ Which drinks contain the most caffeine?

🔥 Which beverages have the highest calories?

🍬 Which drinks contain the most sugar?

📊 How are beverages distributed across categories?

The goal of this project is to turn raw beverage data into meaningful and easy-to-understand insights through interactive visualizations.
